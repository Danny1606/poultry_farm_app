# Poultry Farm Management App 🐓

A web-based application to manage poultry farms: birds, egg production, feed, sales, and reports.  
Built with **Node.js**, **Electron**, and **SQLite** for a clean and efficient farm management experience.

---

## 🚀 Features
- Track bird inventory and egg production
- Record feed usage and sales
- Generate farm reports
- Local SQLite database for storage
- Electron-based desktop interface
